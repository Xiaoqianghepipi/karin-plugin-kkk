### 如何配置？
打开 `config/config/cookies.yaml` 根据需要配置ck<br>
注意冒号后有个空格！

::: code-group
```yaml{2,5,8} [config/config/cookies.yaml]
# 抖音ck
douyin: 此处填上你的抖音ck

# B站ck
bilibili: 此处填上你的B站ck 

# 快手ck
kuaisha: 此处填上你的快手ck
```
:::
